# Release notes for v1.0.0

This is the first public baseline of the Mopedgarage extension.

Highlights:
- one frozen initial migration (`v100`)
- public gallery/search page
- profile and post integration
- ACP settings and custom-field management
- UCP self-management for user bikes
- image uploads with server-side resize

Important:
This release is meant to be the clean starting point for GitHub and future versioned updates.
